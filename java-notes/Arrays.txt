String methods:
length() - length of characters in a sentence
	     String list = "asdfghhfdgsfads";
         System.out.println(list.length()); //answer is 15
 
concat() -  to add two strings together
	  String list = "googoo";
        String lis = " gahgah";
        System.out.println(list.concat(lis)); //answer is googoo gahgah

equals() - to check if two strings are equal to each other because we can't use ==
	String car1 = "toyota";
	String car2 = "camry";
	System.out.println(car1.equals(car2)); //answer should be false
	
	or 
	
	String password = "apple";
	System.out.println(password.equals("apple")); //should be true
	
indexOf() - to check position of letter
	System.out.println(password.indexOf("e"); //answer should be 4
	
charAt() - to get character at a specific index
	System.out.println(password.charAt(4); //answer should be e
	
substring() - extract a portion of a string
	     String name = "baddestboj";
        System.out.println(name.substring(7,10));
